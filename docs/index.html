ğŸš€ Quick Start

1. Clone Repository

git clone https://github.com/YOUR_USERNAME/mybambu-conversational-transfers.git
cd mybambu-conversational-transfers
npm install

2. Configure Environment

Get Wise Sandbox API credentials from sandbox.transferwise.tech

MODE=PRODUCTION
WISE_API_KEY=your-sandbox-key
WISE_PROFILE_ID=your-profile-id
WISE_API_URL=https://api.sandbox.transferwise.tech

3. Setup Claude Desktop

Edit ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "mybambu-claude": {
      "command": "node",
      "args": ["/path/to/dist/server.js"],
      "env": {
        "MODE": "PRODUCTION",
        "WISE_API_KEY": "your-key",
        "WISE_PROFILE_ID": "your-id",
        "WISE_API_URL": "https://api.sandbox.transferwise.tech"
      }
    }
  }
}

4. Build & Test

npm run build
# Restart Claude Desktop
# Say: "Send $100 to Colombia"
