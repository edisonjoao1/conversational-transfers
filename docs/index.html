Frequently Asked Questions

Everything you need to know

ü§î How is this different from traditional money transfer apps?
Traditional apps require multiple screens, form filling, and remembering complex banking details. With conversational transfers, you simply chat naturally with AI. Say "Send $100 to Maria in Mexico" and the AI handles the rest conversationally. It's 10x faster and requires zero learning curve.
üîí Is this secure? How do you handle sensitive data?
We use a reference-based architecture where sensitive data (like full account numbers) are stored server-side only and never sent to AI models. ChatGPT sees only masked data (e.g., "...8952"). We're fully PSD2 compliant and use Wise's regulated API infrastructure. All transfers are executed through Wise's secure banking partners.
üí∞ What are the fees?
Fees are powered by Wise API and vary by corridor, typically $2.99-$4.65 for most transfers. You always see the exact fee before confirming. No hidden charges, no markup on exchange rates. What you see is what you pay.
üåç Which countries can I send money to?
Currently live: Mexico, Colombia, Brazil, United Kingdom, and Europe. In development: 45+ additional countries including Philippines, India, Vietnam, Nigeria, and more. The system is built on Wise API which supports 80+ countries globally.
‚ö° How fast are transfers?
Transfer speeds vary by corridor. Fastest routes (Mexico, Dominican Republic) take ~35 minutes. Most transfers complete within 1-3 hours. Some corridors may take up to 24 hours. You'll always see estimated delivery time before confirming.
ü§ñ Which AI platforms do you support?
Claude Desktop is live and production-ready. ChatGPT Custom GPT is in beta testing with moderation bypass. WhatsApp Business API is planned for Q1 2026. All platforms share the same backend for consistent experience.
üíª Can I integrate this into my own app?
Yes! The entire system is open source on GitHub. Full documentation, TypeScript codebase, deployment guides for Railway/Render. You can fork, modify, and deploy your own version. MIT license.
üè¶ Do I need a Wise account?
To deploy your own version, you'll need Wise API credentials (sandbox for testing, production for real transfers). End users don't need Wise accounts - the system handles everything behind the scenes.
